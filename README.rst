RsSFETCH
========


**NAME**


|
| ``rssfetch`` - RsS Fetcher
|


**SYNOPSIS**


|
| ``rssfetch <opmlfile>``
|


**DESCRIPTION**


``rssfetch`` echo's rss feeds into your IRC channel from 1 OPML file


**INSTALL**


installation is done with pipx

|
| ``$ pip install rssfetch``
|
